Abstract

PILA (PolyTrack Imitation Learning Agent) is an imitation learning agent trained to play PolyTrack by learning directly from recorded human gameplay — not from hard-coded rules or game internals.

Instead of manually programming driving behavior, PILA uses supervised learning to map game states (visual frames) to player actions, allowing it to imitate realistic human driving purely from pixels.

Method

Data Collection

Gameplay is recorded as observations (screen frames) and actions (keyboard/mouse inputs). The system captures:

  • Steering (A/D keys or mouse movement)
  • Throttle/Brake (W/S keys or space)
  • Special actions (Q/E, Shift, Ctrl)

Training

A convolutional neural network is trained using imitation learning. The model minimizes the loss between predicted actions and recorded human actions. Checkpoints are saved every 2 epochs.

Inference

During gameplay, the trained model reads live game frames, predicts steering/throttle/brake inputs, and sends keyboard commands to the game in real-time.

Key Features

End-to-End Pipeline

Complete workflow from data collection to real-time inference in a single codebase.

Learns by Example

No hard-coded logic — learns complex driving styles directly from human gameplay.

Real-Time Performance

Efficient screen capture (dxcam) and streamlined model for low-latency gameplay.

TensorBoard Support

Visualize training progress with loss curves and attention metrics.

Getting Started

Clone & Install

git clone https://github.com/tryfonaskam/pila.git
cd pila
pip install -r requirements.txt

Record Data

python3 loop_datamaker.py

Press F1 to start recording, play the game, then F10 to save.

Train

python3 train.py

Play

python3 play.py

Switch to game window within 5 seconds. Press ESC to stop.

Requirements

  • Python 3.11+
  • CPU or GPU (CUDA recommended for training)
  • PolyTrack game
